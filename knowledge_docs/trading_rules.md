# 交易心法与风控纪律

> 来源：华尔街顶尖交易员经验整理（布鲁斯·柯凡纳等）+ 贵金属专项纪律。
> 本文件已作为知识种子导入系统，规则将参与回测评分与自我进化。

---

## 心态纪律

- 绝不扛单，每次进场前必须预先设定止损点，触及止损即认错离场，无任何例外
- 如果行情触及止损价位，意味着该笔交易的分析出现了错误，必须立刻认错离场
- 预设止损是让交易员安心入睡的唯一办法，也是铁血纪律的核心
- 连续亏损两笔后当天必须立刻停止交易，防止复仇交易进一步扩大损失
- 双亏熔断原则：连续输掉两场交易，当天必须停止所有操作，绝无例外
- 复仇交易是账户爆仓的直接前驱，不得通过更多交易来试图回本
- 感到愤怒、恐惧、过度兴奋或过度自信时，必须立即停止交易并进行自我调节
- 情绪波动是交易禁区，不是灵感来源，任何情绪化状态下禁止开仓
- 使用3分钟重置法则：离开屏幕，深呼吸，记录当前情绪状态后再做决策
- 获利往往源于进场前的耐心准备工作，而非偶然运气
- 盈利后最危险的陷阱是掉以轻心、开始频繁操作，容易将所有盈利回吐
- 连胜后应主动降低仓位，保持敬畏心，不因连胜而盲目加仓
- 每笔交易后详细记录决策依据、执行情况、结果和当时情绪状态
- 定期复盘交易日记，发现自身弱点并持续优化策略

## 风险控制

- 连续亏损时必须不断减小交易仓位，甚至直接停止操作，切勿增仓追回
- 面对亏损时减量经营，绝不情绪化地增加频率或仓位来挽回颓势
- 任何时候切忌满仓操作，必须深刻理解杠杆的双刃剑效应
- 重仓杀进杀出是绝大多数失败交易者的共同特征，必须坚决避免
- 杠杆放大盈利的同时也等比放大亏损，100倍杠杆下价格反向1%本金即归零
- 未平仓头寸的总量必须与账户净值相匹配，不得无视保证金占用比例
- 单笔最大风险不超过账户净值的2%，以此逆运算开仓手数
- 止损单必须在开仓时同步设置，不得开仓后再补设或随意移动止损
- 不得因为看好行情而无视账户剩余保证金的安全边际
- 黄金合约乘数100，1手等于100盎司，价格每涨1美元盈亏100美元
- 外汇USDJPY基础货币是USD，1手等于10万美金面值，不能将手数小等同于风险小

## 宏观大局与顺势

- 黄金价格与美元汇率呈反比关系，美元走弱时金价往往上涨
- 黄金与实际利率呈反向关系，实际利率上升则金价承压
- 黄金在地缘政治紧张时发挥避险功能，避险情绪升温利好金价
- 金价通常与国际油价呈正向变动，通胀预期联动
- 央行购金量增加对金价形成长期正向支撑
- 宏观大时间框架的趋势由机构和市场情绪驱动，逆势操作具有极高风险
- 等待趋势消退的明确信号出现后，再考虑反趋势头寸，不得轻易逆势
- 在金价上升趋势中买入往往正确，但必须极度警惕顶部转势风险
- 不能只看绝对价格，必须判断当前处于大牛还是大熊的总体趋势之中
- RSI超过75形成超买且出现多周期背离时，应高度警惕顶部转势
- 美国非农和CPI等重大事件发布前后，应以"预期差"方向为核心判断依据
- 出手分级为"不宜出手"时，系统已判断当前为逆势或高风险场景，必须严格遵守

## 知行合一

- 有策略地等待高概率机会，而非频繁寻找交易机会
- 进场前必须有完整的分析逻辑，不得因为"感觉"而随意开仓
- 盈亏比低于1:1.5的交易信号，不值得冒险执行
- 不确定方向时，宁可空仓观望，也不要强行押注
- 已设定止盈止损后，不得因短期价格波动而随意干预，让规则自动执行
